1. What does the acronym URL stand for?

  • Universal Resource Link
  • Uniform Resource Locator
  • Universal Resource Location
  • Unique Resource Locator

Ans: Uniform Resource Locator


2. What is the primary protocol used to transfer data on the World Wide Web?

  • FTP
  • TCP
  • HTTP
  • DNS

Ans: HTTP


3. What is the name of the first successful internet search engine?

  • Google
  • Bing
  • Yahoo
  • Archie

Ans: Archie


4. What is the name of the technology that allows for the secure exchange of information over the internet?

  • VPN
  • SSH
  • SSL
  • SCP

Ans: SSL


5. What is the name of the technology used to compress data for faster transmission over the internet?

  • Gzip
  • Zip
  • RAR
  • Tar

Ans: Gzip


6. What is the name of the first web browser?

  • Chrome
  • Firefox

7. What is the name of the browser engine used by Google Chrome?

  • Blink
  • Gecko

  • EdgeHTML

8. What is the name of the web browser that utilizes the Webkit engine?

  • Chrome
  • Firefox
  • Safari
  • Edge

9. What is the main characteristic of a static website?

  • The content is dynamic and changes frequently
  • The content is stored in a database and generated dynamically
  • The content is fixed and does not change frequently
  • The website is only accessible through a specific browser

Ans: The content is fixed and does not change frequently


10. What are the disadvantages of using a static website?

  • Limited functionality, No database integration, No user interaction
  • High cost, Complex to maintain, Slow loading times
  • Limited scalability, Lack of security, Limited speed
  • No flexibility, No customization, No interactivity

Ans: Limited functionality, No database integration, No user interaction


11. What is the name of the most popular dynamic site generator?

  • Jekyll
  • WordPress

12. What is the name of the framework that allows you to build dynamic websites using React?

Profile Building Quiz(16 Jan'23)

1. What is the primary goal of UI/UX design?

  • To make a product aesthetically pleasing.
  • To make a product easy and enjoyable to use.
  • To make a product popular among users.
  • To make a product cheap to produce.

Ans: To make a product easy and enjoyable to use.


2. What is the importance of wireframing in the design process?

  • It helps to define the layout and functionality of a product before it is built.
  • It helps to decide on the color scheme and typography of a product.
  • It helps to test the performance of a product.
  • It helps to decide on the target market of a product.

Ans: It helps to define the layout and functionality of a product before it is built.


3. What does HTML stand for?

  • Hypertext Markup Language
  • Hyperlink Transfer Protocol
  • Hypertext Transfer Protocol Secure
  • Hyperlink Markup Language

Ans: Hypertext Markup Language


4. Which of the following is NOT a JavaScript framework?

  • AngularJS
  • ReactJS
  • Ruby on Rails
  • Vue.js

Ans: Ruby on Rails


5. What is the primary use of CSS?

  • To create the layout and design of a webpage
  • To add functionality to a webpage
  • To create and manipulate database connections
  • To handle server-side logic

Ans: To create the layout and design of a webpage


6. What is the primary use of a database in web development?

  • To store and retrieve data for a web application
  • To handle server-side logic
  • To create the layout and design of a webpage
  • To add functionality to a webpage

Ans: To store and retrieve data for a web application


7. What is the purpose of an API in web development?

  • To provide a way for different applications to communicate with each other
  • To handle client-side logic
  • To create the layout and design of a webpage
  • To add functionality to a webpage

Ans: To provide a way for different applications to communicate with each other


8. What is the role of a backend developer?

  • To create and maintain the user interface and user experience of a web or mobile application
  • To create and manage the server-side of an application, including the database, server-side logic, and APIs
  • To ensure that a website looks the same on all devices
  • To create and manage the network infrastructure of an application

Ans: To create and manage the server-side of an application, including the database, server-side logic, and APIs


9. What is the primary purpose of a portfolio in the field of design and technology?

  • To showcase the individual's skills and experience to potential employers or clients
  • To document the individual's learning process and progress
  • To provide a platform for the individual to share their work with the public
  • All of the above

Ans: All of the above


10. Why is a portfolio important for a job seeker?

  • It demonstrates their skills and experience to potential employers
  • It showcases their attention to detail and ability to showcase their work in a professional manner
  • It highlights their ability to communicate their design process and decisions
  • All of the above

Ans: All of the above
