Base Building Quiz

1. What does the acronym URL stand for?

a. Universal Resource Link

b. Uniform Resource Locator

c. Universal Resource Location

d. Unique Resource Locator

Ans: b. Uniform Resource Locator


2. What is the primary protocol used to transfer data on the World Wide Web?

a. FTP

b. TCP

c. HTTP

d. DNS

Ans: c. HTTP


3. What is the name of the first successful internet search engine?

a. Google

b. Bing

c. Yahoo

d. Archie

Ans: d. Archie


4. What is the name of the technology that allows for the secure exchange of information over the internet?

a. VPN

b. SSH

c. SSL

d. SCP

Ans: c. SSL


5.What is the name of the technology used to compress data for faster transmission over the internet?

a. Gzip

b. Zip

c. RAR

d. Tar

Ans: a. Gzip


6. What is the name of the first web browser?

a. Chrome

b. Firefox

c. Internet Explorer

d. Mosaic

Ans: d. Mosaic

7.What is the name of the browser engine used by Google Chrome?

a. Blink

b. Gecko

c. Webkit

d. EdgeHTML

Ans: a. Blink


8. What is the name of the web browser that utilizes the Webkit engine?

a. Chrome

b. Firefox

c. Safari

d. Edge

Ans: c. Safari


9. What is the main characteristic of a static website?

a. The content is dynamic and changes frequently

b. The content is stored in a database and generated dynamically

c. The content is fixed and does not change frequently

d. The website is only accessible through a specific browser

Ans: c. The content is fixed and does not change frequently


10. What are the disadvantages of using a static website?

a. Limited functionality, No database integration, No user interaction

b. High cost, Complex to maintain, Slow loading times

c. Limited scalability, Lack of security, Limited speed

d. No flexibility, No customization, No interactivity

Ans: c. Limited functionality, No database integration, No user interaction


11. What is the name of the most popular dynamic site generator?

a. Jekyll

b. WordPress

c. Drupal

d. Ghost

Ans: b. WordPress


12. What is the name of the framework that allows you to build dynamic websites using React?

a. Next.js

b. Gatsby

c. Nuxt.js

d. Ember.js

Ans: a. Next.js
