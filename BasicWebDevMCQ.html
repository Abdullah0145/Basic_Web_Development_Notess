Base Building Quiz(15 Jan'23)

1. What does the acronym URL stand for?

a. Universal Resource Link

b. Uniform Resource Locator

c. Universal Resource Location

d. Unique Resource Locator

Ans: b. Uniform Resource Locator


2. What is the primary protocol used to transfer data on the World Wide Web?

a. FTP

b. TCP

c. HTTP

d. DNS

Ans: c. HTTP


3. What is the name of the first successful internet search engine?

a. Google

b. Bing

c. Yahoo

d. Archie

Ans: d. Archie


4. What is the name of the technology that allows for the secure exchange of information over the internet?

a. VPN

b. SSH

c. SSL

d. SCP

Ans: c. SSL


5.What is the name of the technology used to compress data for faster transmission over the internet?

a. Gzip

b. Zip

c. RAR

d. Tar

Ans: a. Gzip


6. What is the name of the first web browser?

a. Chrome

b. Firefox

d. Mosaic

d. EdgeHTML

8. What is the name of the web browser that utilizes the Webkit engine?

a. Chrome

b. Firefox

9. What is the main characteristic of a static website?

a. The content is dynamic and changes frequently

b. The content is stored in a database and generated dynamically

c. The content is fixed and does not change frequently

d. The website is only accessible through a specific browser

Ans: c. The content is fixed and does not change frequently


10. What are the disadvantages of using a static website?

a. Limited functionality, No database integration, No user interaction

b. High cost, Complex to maintain, Slow loading times

c. Limited scalability, Lack of security, Limited speed

d. No flexibility, No customization, No interactivity

Ans: c. Limited functionality, No database integration, No user interaction


11. What is the name of the most popular dynamic site generator?

Profile Building Quiz(16 Jan'23)

1. What is the primary goal of UI/UX design?

a. To make a product aesthetically pleasing.

b. To make a product easy and enjoyable to use.

c. To make a product popular among users.

d. To make a product cheap to produce.

Ans: b. To make a product easy and enjoyable to use.


2. What is the importance of wireframing in the design process?

a. It helps to define the layout and functionality of a product before it is built.

b. It helps to decide on the color scheme and typography of a product.

c. It helps to test the performance of a product.

d. It helps to decide on the target market of a product.

Ans: a. It helps to define the layout and functionality of a product before it is built.


3. What does HTML stand for?

a. Hypertext Markup Language

b. Hyperlink Transfer Protocol

c. Hypertext Transfer Protocol Secure

d. Hyperlink Markup Language

Ans: a. Hypertext Markup Language


4. Which of the following is NOT a JavaScript framework?

a. AngularJS

b. ReactJS

c. Ruby on Rails

d. Vue.js

Ans: c. Ruby on Rails


5. What is the primary use of CSS?

a. To create the layout and design of a webpage

b. To add functionality to a webpage

c. To create and manipulate database connections

d. To handle server-side logic

Ans: a. To create the layout and design of a webpage


6. What is the primary use of a database in web development?

a. To store and retrieve data for a web application

b. To handle server-side logic

c. To create the layout and design of a webpage

d. To add functionality to a webpage

Ans: a. To store and retrieve data for a web application


7. What is the purpose of an API in web development?

a. To provide a way for different applications to communicate with each other

b. To handle client-side logic

c. To create the layout and design of a webpage

d. To add functionality to a webpage

Ans: a. To provide a way for different applications to communicate with each other


8. What is the role of a backend developer?

a. To create and maintain the user interface and user experience of a web or mobile application

b. To create and manage the server-side of an application, including the database, server-side logic, and APIs

c. To ensure that a website looks the same on all devices

d. To create and manage the network infrastructure of an application

Ans: b. To create and manage the server-side of an application, including the database, server-side logic, and APIs


9. What is the primary purpose of a portfolio in the field of design and technology?

a. To showcase the individual's skills and experience to potential employers or clients

b. To document the individual's learning process and progress

c. To provide a platform for the individual to share their work with the public

d. All of the above

Ans: d. All of the above


10. Why is a portfolio important for a job seeker?

a. It demonstrates their skills and experience to potential employers

b. It showcases their attention to detail and ability to showcase their work in a professional manner

c. It highlights their ability to communicate their design process and decisions

d. All of the above

Ans: All of the above
